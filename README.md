# Projektwoche

## Installation

1. Clone git repo

```bash
git clone https://github.com/Yeet195/Projektwoche
```

---

2. Change directory to project

```bash
cd Projektwoche
```

---

3. Make start.sh executable

```bash
sudo chmod +x ./start.sh
```

---

4. Run start.sh

```bash
sudo bash ./start.sh
```
